    How it works
    
      
    A .docx file is a ZIP archive of XML parts. When Word reports "unspecified error", the
      most common cause is truncated or malformed XML in word/document.xml — or a damaged ZIP
      container that prevents Word from even opening the file. This tool:
    
      
      
        
    1. Unpacks the DOCX in memory using JSZip — and, if JSZip refuses the archive or can't decompress a
        corrupt DEFLATE entry, falls back to the Immortal Inflater: a fault-tolerant DEFLATE
        decoder (from
        Universal-File-Repair-Tool)
        that continues past corrupted blocks and returns partial data instead of failing.
      2. 
        
    2. Validates word/document.xml with the browser's XML parser.
      3. 
        
    3. If invalid or only partially decoded, locates the last well-formed paragraph
        (</w:p>) and rebuilds a clean document with proper closing tags.
      4. 
        
    4. Repacks the archive and lets you download the recovered .docx or extract plain text.
      5. 
      
    
      
    Recovery is best-effort. For severely damaged files, save the plain-text export and rebuild
      formatting manually.

    • iOS / iPadOS: Open in Safari, then Share → Add to Home Screen.
